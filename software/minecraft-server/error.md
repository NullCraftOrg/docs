# 异常报错

{% hint style="info" %}
此处用于存储软件异常错误的解决方案
{% endhint %}

寻找异常时请通过 异常信息 进行对比，软件报错内的 异常信息 与 此文档内的 异常信息。

### 手动编目：编号-0x001-A

**异常信息：**的类型初始值设定项引发异常。

**异常信息：**无法加载计数器名称数据，因为从注册表读取的索引""无效、

**异常原因：**以上两个异常都代表你的 性能监视器 数据出现了异常，导致软件无法正确获取。

**解决方案：**使用管理员身份运行 `cmd` 输入 `lodctr /r` 提示恢复成功 再次打开软件即可正常
