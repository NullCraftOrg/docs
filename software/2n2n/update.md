### v3.0.0.105 (2025-01-05)

{% hint style="info" %}
**提示**

更新此版本需要更新 .NET 至 **.NET8**
{% endhint %}

<mark style="color:blue;">**`[更新]`**</mark>

* .NET 版本升级至 .NET8
* 整体框架 80%以上代码进行了重构
* 档案的快速分享/导入功能
* 档案只读状态(需要手动设置)
* 节点现在有更好的管理窗口了
* 更多的组网状态通知
* 移除了一些不必要的功能

<mark style="color:orange;">**`[优化]`**</mark>

* 档案读取减少了读写次数(2-3次 -> 1次)
* 档案选择读取/写入速度提升
