# 文档状态标记说明

{% hint style="info" %}
为了简洁并可视化的显示每个文档页面的状态，在这里列出不同状态下的标记方法和预览。
{% endhint %}

## 文档的更新状态

{% hint style="info" %}
文档的更新状态使用 Emoji 表情作为显示，主要是因使用文字标记可能导致过长无法正常显示。
{% endhint %}

### 状态：已完成

无任何状态图标

### 状态：正在更新

使用 Emoji 预览：🚧

### 状态：暂停更新

使用 Emoji 预览：🚫

### 状态：已被废弃

使用 Emoji 预览：❌
