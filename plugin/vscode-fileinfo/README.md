## Fileinfo - 一个 VSCode 插件

目前它可以显示二进制文件信息。

### Use / 使用

安装它或者在 VSCode 拓展中搜索 fileinfo 安装即可启用

### Features / 功能

一个练手的项目， 它可以在右下角底部状态栏中显示当前打开的二进制文件的大小(KB)。

### Download / 下载

* **`[👍]`** VSCode 插件市场 中 搜索 `FastChen.fileinfo` 安装，可以自动同步更新
